# ⛺ Web de Camping

> **Landing Page Moderna para un Complejo de Camping.**

---

## 📖 Descripción
Este proyecto es una **Landing Page** diseñada para "Camping", un espacio donde conectar con la naturaleza de forma cómoda y amena.
El diseño busca transmitir **calma, aventura y conexión**, mediante una estética limpia que evoca la naturaleza, fotografías inmersivas y un flujo de navegación intuitivo. Su objetivo principal es invitar al usuario a disfrutar de una escapada relajante.

### Características Principales y Técnicas:
*   ✅ **Diseño Responsivo:** Se adapta de forma fluida y perfecta a celulares, tablets y opciones de escritorio.
*   ✅ **Navegación Eficiente:** Scroll suave y animado entre cada sección.
*   ✅ **Estética Natural:** Paleta de colores orgánicos y espaciados amplios para mayor descanso visual.
*   ✅ **Alta Performance:** Construido en Vite para garantizar tiempos de carga mínimos.
*   ✅ **Componentes Modulares:** Estilos mantenidos en Vanilla CSS organizados por componente para una fácil escalabilidad en módulos.
*   ✅ **Arquitectura React:** Manejo de estado efectivo mediante el uso de **Hooks** para componentes interactivos (menús móviles, modales, alertas).
*   ✅ **Imágenes Optimizadas:** En formato webp, para mejorar los tiempos de carga.

---

## 🛠️ Tecnologías Utilizadas

*   HTML5
*   CSS3 (Vanilla CSS, variables globales)
*   JavaScript (ES6+)
*   React 19
*   Vite

## 📂 Estructura del Proyecto

```bash
src/
├── 📂 assets/          # Imágenes de naturaleza y galería (formato .webp)
├── 📂 components/      
│   ├── 📂 Header/           # Barra de navegación principal y menú móvil
│   ├── 📂 Hero/             # Sección de presentación y llamado a la acción
│   ├── 📂 About/            # Información sobre el complejo y la filosofía
│   ├── 📂 Services/         # Servicios ofrecidos y comodidades
│   ├── 📂 Gallery/          # Galería de imágenes inmersiva
│   ├── 📂 Location/         # Mapa e información sobre cómo llegar
│   ├── 📂 Reviews/          # Testimonios de campistas felices
│   ├── 📂 Contact/          # Formulario de contacto
│   ├── 📂 FloatingWhatsApp/ # Botón flotante para mensajería directa
│   └── 📂 Footer/           # Cierre del sitio y redes sociales
├── 📂 index.css        # Configuración de variables CSS y estilos base
└── 📄 main.jsx         # Punto de entrada de la aplicación React
```
